# 引言

在教程进行中，我们发现了许多问题。有的同学觉得我们的课程跳跃度太大，有很多语法并没有涉及。首先呢，作为WKU-SRA的第一次线上暑培，我们的培训经验并不充足，
且WKU-SRA编者均在业余时间进行编写，各位编者有自己的时间安排。WKU-SRA飞跃手册非盈利的性质也导致编者们并没有任何报酬。
作为一门编程入门课，我们更希望和大家介绍编程中的一些编程哲学，编程之禅，拓展大家的视野，故我们并没有很多时间为大家整理基础知识和语法。

综上，我们决定在这里为大家整理一些Python的基础知识，大家可以对照判断自己对这些知识点的理解程度，跳转对应文档进行具体学习。
以下内容不做引用说明，内容均改编于[Python 官方中文教程文档](https://docs.python.org/zh-cn/3/tutorial/)
