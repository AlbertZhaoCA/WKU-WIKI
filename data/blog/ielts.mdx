# 听力技巧

🌟地图题  
1.沉浸式带路题：仿佛你在里面走 - 先找entrance和road以及“你”在哪里（starting point），因为会先从这开始讲 - 有方向，记住左右且标注路线2.上帝视角带路题：你是局外人 - NSWE在图里标好

🌟填空题

1. 先看word limit
2. 预判答案
   词性大部分为n/adj 少数adv/v（40题里最多两个）  
   3.不带答案的句子也要圈画关键词用于把握原文脉络，重点关注衔接词  
   4.听力考试结束统一检查时态语法和拼写

🌟选择题

1. 一定要在多余的时间里提前看选择题
2. 在选择题给的看题时间里复习一遍题干和选项（时间不够的话优先看题干）
3. 听力正式开始前记住每道题干关键词（听题重点）
4. 做题时始终保持对连续两道题（正在放的和下一道）的敏感度，听到下一道的关键词果断放弃上一道

🌟匹配题 最难题型

1. 注意看形容词并听反义词
2. 有并列关系的选项原文大概率会一句说其中的一部分合起来才是答案

🌟tips

1. 单复数形式与原文一致
2. 答案首字母大小写都行
3. 月份和星期不可以简写
4. 建议不加th st等角标
5. 替换词选择
   优先级：n.adj.adv.疑问词.v  
   注意：否定.比较级.最高级.强调词

# 阅读技巧。

1. 必须做很多题对雅思有了充分了解后再去学习题型解题思路以及替换词等。
2. 专项训练没有用，因为一套雅思阅读的题型很综合，不可能把所有的精力用在一种题型上，需要培养的就是对各个题型同时出现的灵敏度
3. 拿到一篇阅读题应该做什么 word limit

🌟浏览全部模块并判断模块内是顺序还是乱序  
顺序：记住每个部分前2～3道  
乱序：对每个题号关键词进行记忆且读完一段回来对题干  
🌟判断模块内的题在原文中的答案是集中还是分散  
🌟判断模块间关系

先后：先做前面  
并列：每个都顾及

4. tips  
   🌟同义替换或者填空题目有可能只是对一小部分细节进行考察  
   🌟概括类题型看本段的大篇幅笔墨写了什么  
   🌟选择题经常需要整体理解 千万别钻牛角尖  
   🌟判断对错题找到题干的本质 看原文对本质的阐述 其他不重要  
   🌟判断对错题有时候和原文大相径庭非常离谱，这时候应该结合多段原文进行判断，大概率是对的：注意从属关系

# 口语技巧

考试打分解析

1. 目标分数备考策略
   目标分6.5：重心放在P1 P2  
   打分关键：背题感/串题感  
   目标分7：重心放在P3  
   打分关键：逻辑性/组织性  
   目标分7.5：P3论点质量/新颖  
   目标分8+：交流感/用英语思维的能力

2. 口语四小分精确剖析  
   流利度：短期容易提分 要求你把题库说至少3遍  
   发音：短期放弃 native语音语调更重要  
   词汇：短期极易提分！！！把低级词汇都替换掉家人们别beautiful interesting happy了  
   语法：短期放弃 别犯大错  
   综合分析：建议短期一起提词汇和流利度  
   常见问题回答

- Q1：存不存在单项小分很高其它很低的情况？  
  A1：不存在，因为打分虽然分为四项，语言是一个整体
- Q2：那单项提分是什么意思？  
  A2：比如你的正常整体水平应该是6分，而因为词汇高级多出0.5小分，因此口语总成绩向上取整变成6.5
- Q3：我口语流利，P2说不满2分钟或者脑子里啥都没有胡扯影响大吗？  
  A3 巨巨巨巨巨巨大！流利度和发音哪怕你是八分水平，因为你内容辣鸡，每个都减0.5-1左右（因为本人亲自尝试过模考瞎说） 雅思如果考中文，你也不能在1分钟之内就题目说两分钟的小故事，因为有的题真的没得可说！如果你不信，一分钟之内准备时间给我描述一件你爱穿的衣服说两分钟（正常人表示emmmm？） 这里最说明语言是一个整体！你操着一口纯正的英文流利的说一些低级没用的话考官默认你没有能力用同样纯正流利的表达去说更复杂的

关于模板的使用

1. 7+杜绝！！P123都不准用！
2. 别听机构给你串的题套的模板 你要被发现用模板就没了
3. 考官考了十万八千个考生 你说的是真是假 是不是自己的 还能瞒得过去？
4. 雅思考试人与人对话，对面不是考官，是一个平等的和你交流的人。正常说话谁每个问题都有模板？  
   雅思备考和考场须知  
   考试自己估计小分：流利度8 发音8 词汇8 语法7
5. 怎样伪造无准备感
6. 怎样准备P2
7. 怎样准备P3
8. 到底哪些地方影响得分

- 到底可不可以偏题？如果可以，什么情况下可以？
- 说错了要重复还是自我修正
- 用词不当会咋样
- 车轱辘会咋样
- 考官的问题应当怎样处理

5. 怎样保持平等交流感
6. 怎样讲自己的故事
7. 怎样在考场上增加对考试的掌控权
